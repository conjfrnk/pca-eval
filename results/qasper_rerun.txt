============================================================
KEY OBSERVATIONS
============================================================

1. BEST CONFIG (pre-trained): Base + decompose + passage-scoring at t=0.15 = 69.5% F1
   - This is the best achievable without fine-tuned models

2. Decompose + passage-scoring is critical: +8.7pp F1 (base), +6.2pp F1 (large) over plain NLI
   - These features dramatically improve ANSWERABLE accuracy (25.6% -> 62.2% for base)
   - UNANSWERABLE accuracy remains 100.0% across all configs

3. Threshold t=0.15 > t=0.5: +1.5pp F1 (base pipeline), +1.7pp F1 (large pipeline)
   - Lower threshold correctly classifies more ANSWERABLE examples
   - Legitimate because 90.5% of the dataset is ANSWERABLE

4. Base > Large (pre-trained): Base outperforms Large in all pipeline configs
   - Base + pipeline t=0.15: 69.5% vs Large + pipeline t=0.15: 68.7% (-0.8pp)
   - This reverses with fine-tuning (fine-tuned large = 88.2%)

5. Fine-tuning gap: ~19pp between pre-trained best (69.5%) and fine-tuned best (88.2%)
   - Fine-tuned large-512 with threshold optimization was the previous best
   - Cannot reproduce 88.2% without fine-tuned model checkpoint
