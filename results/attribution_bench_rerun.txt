=========================================
KEY FINDINGS
=========================================

1. BEST OOD RESULT (pre-trained): DeBERTa-v3-large at t=0.30 achieves 71.9% sklearn macro-F1
   - This is 6.1pp below GPT-4 w/o CoT (78.0%) and 10.0pp below fine-tuned GPT-3.5 (81.9%)
   - The gap is significant and expected: pre-trained NLI models have not been fine-tuned on
     attribution data

2. ID vs OOD gap: ~9-10pp macro-F1 advantage on OOD vs ID
   - OOD includes HAGRID examples which our models are better calibrated for
   - ID datasets (ExpertQA, Stanford-GenSearch) have longer, more complex claims

3. ATTRIBUTABLE recall is the bottleneck:
   - ID: 37-43% ATTRIBUTABLE accuracy (model is very conservative)
   - OOD: 53-59% ATTRIBUTABLE accuracy (better but still conservative)
   - NOT_ATTRIBUTABLE: 85-92% accuracy (model excels at detecting unsupported claims)

4. Model size helps on OOD (+2.7pp macro-F1) but barely on ID (+1.8pp):
   - Large model trades precision for recall (more balanced per-class accuracy)

5. MiniCheck fallback helps on ID (+1.8pp accuracy) by boosting ATTRIBUTABLE recall

6. Decompose-evidence HURTS performance (-2.5pp report F1) - evidence chunking fragments
   context needed for whole-claim attribution

7. Strategy variants (multi-signal, weakest-link, majority) show minimal difference
   from whole strategy on this benchmark - claims are mostly single sentences

8. Threshold sensitivity is LOW: all thresholds in [0.3, 0.8] produce similar results

NOTE: Previous best result of 81.5% macro-F1 used domain-specific XGBoost trained on
AttributionBench training data with fine-tuned DeBERTa-large-512 features. Those
fine-tuned models are not present in this environment, so XGBoost strategy could not
be evaluated. The results above reflect pre-trained model performance only.
